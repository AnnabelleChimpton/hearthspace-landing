why i'm building this

Six months ago my friend posted this perfect photo from our beach trip. We all wanted to save it. We spent twenty minutes scrolling through Discord trying to find it.

We never found it.

And I just thought: this is stupid. We use these tools for friendship but they're not built for friendship. They're built for engagement metrics and ad revenue and scale we don't need.

Discord is great if you're running a gaming server for 500 people. But for eight friends who want to remember good times and not lose track of plans? It's chaos.

So I'm building something different. For small groups who actually like each other. No algorithm. No ads. No engagement optimization. Just a place that works.
